# 30-operations
Lean Healthcare. O **padrao** (standard work) e definido uma unica vez; cada
unidade guarda apenas seus **especificos locais** — zero duplicacao entre unidades.
- `value-streams/` fluxo de valor do paciente · `standard-work/` padrao
- `network/` rede de unidades e regioes (expansao nacional)
