# 90-meta
Fonte unica de **padroes**, **templates**, **glossario**, **catalogo** e **indice de ADRs**.
Mudar um padrao aqui exige ADR.

- `standards/` — convencoes (nomes, frontmatter, docs, contratos, dados)
- `templates/` — modelos (ADR, dominio, SOP, protocolo, iniciativa)
- `glossary/` — linguagem ubiqua canonica
- `catalog/` — registro legivel por maquina (dominios, indicadores)
- `adr-index/` — indice global de decisoes
- `diagrams/` — arquitetura em C4
