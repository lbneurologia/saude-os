# Catalogo de indicadores
Cada KPI tem definicao unica: formula, fonte, dono, meta.
Exemplos: NPS, taxa de no-show, LTV, tempo de espera, aderencia a protocolo.
Dashboards e pipelines vivem em `20-platform/data`.
