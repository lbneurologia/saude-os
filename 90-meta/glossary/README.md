# Glossario (linguagem ubiqua)
Um termo, uma definicao canonica. Termos de dominio ficam no glossario do dominio.

- **Paciente ativo** — (definir criterio unico, ex.: com contato nos ultimos 12 meses).
- **Unidade** — instancia fisica/operacional da clinica dentro da rede.
- **Contrato de interface** — acordo versionado de comunicacao entre dominios.
- **PHI** — informacao de saude identificavel do paciente (nunca no repo).
