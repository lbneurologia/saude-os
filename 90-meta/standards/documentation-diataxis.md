# Diataxis
- **Tutorial** — aprendizado guiado (passo a passo).
- **How-to** — resolver uma tarefa especifica.
- **Referencia** — descricao precisa (fatos, contratos).
- **Explicacao** — contexto e porque (inclui ADRs).
Cada documento declara seu tipo em `tags`.
